## Lab 4: Advanced OpenGL Control

### Keyboard Controls

| Key | Action |
|-----|--------|
| **Arrow Keys** (↑↓←→) | Move the shape on screen |
| **R** | Toggle between triangle and rectangle |
| **C** | Cycle color (Original → Red → Green → Blue → Yellow) |
| **T** | Cycle scale size (0.4x → 0.6x → ... → 2.0x) |
| **Space** | Hold for wireframe mode |
| **Esc** | Close the window |
